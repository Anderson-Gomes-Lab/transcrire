# transcrire

Petit script shell pour transcrire des fichiers audio et vidéo en local avec whisper.cpp.

## Prérequis

- macOS
- Homebrew
- ffmpeg
- whisper-cpp
- un modèle Whisper dans `~/whisper-models`

## Utilisation

```bash
~/transcrire.sh "/chemin/vers/fichier.m4a"
~/transcrire.sh "/chemin/vers/fichier.mp4"
